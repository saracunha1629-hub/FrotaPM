# FrotaPM - Sistema de Gestão de Frota

Sistema web para gerenciamento da manutenção das viaturas da Polícia Militar de Blumenau.

## Recursos

- Cadastro e acompanhamento de viaturas
- Plano de manutenção preventiva
- Ordens de serviço
- Histórico de manutenção
- Rastreamento em tempo real com Google Maps
- Indicadores de desempenho (KPIs)
- Geração de relatórios (PDF/Excel)
- Controle de acesso por roles

## Tecnologias

- Front-end: HTML5, CSS3, JavaScript
- Back-end: PHP
- Banco de dados: MySQL
- Maps API: Google Maps

## Instalação

1. Clone o repositório
2. Configure o banco de dados (veja `database/schema.sql`)
3. Configure as credenciais em `config/config.php`
4. Acesse `http://localhost/FrotaPM`
